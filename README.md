# 🏏 Virat Kohli Career Analytics Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Virat Kohli's batting performance across different cricket formats. The dashboard enables users to explore career statistics through dynamic filters and visualizations.

---

## 🎯 Objective

The objective of this project is to transform raw cricket statistics into meaningful insights using interactive dashboards and data visualization techniques.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Microsoft Excel
- Data Visualization
- DAX
- Power Query

---

## 📊 Dashboard Features

- Format-wise analysis (ODI, Test, T20I)
- Year-wise performance analysis
- Interactive year range filter
- Total Runs
- Innings Played
- Batting Average
- Strike Rate
- Boundary Distribution (4s & 6s)
- Runs scored by year
- Interactive visuals and slicers

---

## 📈 Key Insights

- Compare Virat Kohli's performance across formats.
- Analyze yearly batting trends.
- Observe changes in strike rate and batting average.
- Explore boundary distribution.
- View overall career performance through KPI cards.
